#📱 Fit-Fresh
자바 + php_사용자 식습관 진단 어플

## 프로젝트 소개
Fit-Fresh는 사용자의 식습관을 진단하는 건강진단 어플입니다.
간단한 설문조사를 통해 한끼 식사로 섭취하는 영양소를 파악할 수 있고 부족한 성분이 무엇인지 알려줍니다.
또한 결과를 통해 심근경색, 고혈압등 해당되거나 걸릴 수 있는 만성질환을 알려주고 이에 좋은 음식들을 소개해줍니다.

##📌 주요기능
1.로그인 / 회원가입
-사용자 인증을 위해 서버(php)와 통신하여 로그인 처리 및 회원가입 처리(get/post)
-입력값 검증 및 로그인 성공/실패에 따른 결과 구현
-회원가입시 아이디 중복 및 성공/실패 결과 구현
2.사용자 식습관 진단
-사용자의 하루 식습관을 진단할수 있는 설문지를 통해 식습관 데이터 수집
-서버와 통신하여 데이터베이스에 사용자의 영양소(탄수화물 단백질 지방)수치 저장
-평균 사용자 데이터와 비교하여 개선방향 제공 기능 구현
3.사용자 만성질환 진단
-사용자의 나이와 bmi지수 및 설문조사를 통해 건강상태 분석 기능 구현
-만성질환들 중 특정 질환 위험도를 판단하여 경고 및 개선방안 구현



##🛠️ 개발환경
IDE : Android Studio, Visual Studio Code
프론트엔드 : JAVA
백엔드 : PHP
데이터베이스 : MYSQL
기타 : phpMyAdmin
